# Time Management Website

This is a time management website that I am making for my school project.

I want the website to help users organise tasks and manage their time. I am still
learning web programming, so I am building the website one small part at a time.

## What I am using

- Flask for the backend
- HTML for the page structure
- CSS for the page style
- JavaScript for future interactions

## Current progress

The project is currently being set up. I am starting with the basic page layout
and will add more features later.

## How I am working

I am learning while I build this project. I use AI to help me understand code,
but I also check the code and learn what each part does. I am saving small changes
